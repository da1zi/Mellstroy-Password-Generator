## 🎯 О проекте

**Mellstroy Password Generator** - это мощный и удобный генератор паролей с графическим интерфейсом. Создавайте надежные пароли за секунды с возможностью выбора типов символов и быстрого копирования в буфер обмена.

### ✨ Возможности

- ✅ Генерация паролей любой длины
- 🔤 Выбор типов символов (буквы, цифры, символы)
- 📋 Быстрое копирование в буфер обмена
- 🎨 Стильный темный интерфейс
- ⚡ Мгновенная генерация
- 🔒 Надежная энтропия через `random` модуль
